# 频道

## 概念科普

### 什么是频道？

**频道（Channel）** 是 JiuwenSwarm 的消息接入层，作为统一的消息中枢，负责连接各种外部聊天平台（如飞书、钉钉、企业微信、Telegram 等）。它实现了以下核心功能：

| 能力 | 说明 |
|------|------|
| **消息接入** | 接收来自不同平台的消息，统一格式后送入 JiuwenSwarm 处理 |
| **多端同步** | 同一套 JiuwenSwarm 服务可同时接入多个平台，消息互通 |
| **身份识别** | 为每个平台的用户建立唯一标识，维护独立的对话上下文 |
| **安全隔离** | 不同频道的消息流相互隔离，支持白名单、权限控制等 |

### 频道、网页对话与智能体的关系

三者构成了 JiuwenSwarm 的完整对话体系，关系如下：

- **频道（Channel）**：消息接入层，负责接收来自不同平台的消息并统一格式。除了飞书、小艺等外部频道，**网页对话（Web UI）本身也是一个内置频道**。
- **智能体（Agent）**：消息处理引擎，所有频道接入的消息最终都由智能体进行理解、推理和响应。
- **网页对话**：内置的 Web 频道，提供可视化界面供用户直接与智能体交互。


### 简介

JiuwenSwarm 的​**频道**​，是您与不同聊天平台对话的**渠道**。JiuwenSwarm 已经实现了**鸿蒙小艺、飞书**等 channel 的无缝接入，并持续拓展至更多元渠道。可以直接通过 **飞书**、**鸿蒙生态终端的小艺 app** 与 JiuwenSwarm 对话。

### 数字分身

JiuwenSwarm 在**飞书**和**企业微信**频道中支持**群聊数字分身**功能。开启后，机器人将在群聊中作为指定用户的"数字分身"运作——自动识别群聊中与该用户相关的消息并代为回复，以该用户的第一人称口吻进行对话。对于待办、提醒等个人行动类回复，数字分身会自动私发给用户本人，同时在群内补发一条简短确认。无关消息则自动过滤，不消耗 Agent 资源。

该功能默认关闭，开启方式详见下方各频道的配置说明。

### 配置方式

本教程以**网页端配置**为主，后续各频道的接入说明均基于网页操作演示。

如需手动配置，可编辑 `config.yaml` 文件（默认路径：`~/.jiuwenswarm/config/config.yaml`），将对应频道设为 `enabled: true` 并填写鉴权信息，保存后自动重载。

### 频道特性差异

不同频道在私聊/群聊支持、触发方式、权限要求等方面存在差异：

| 特性 | 说明 |
|------|------|
| **私聊/群聊** | 部分频道仅支持私聊（如小艺），部分同时支持私聊和群聊（如飞书、企业微信） |
| **触发方式** | 私聊中通常直接对话即可触发；群聊中可能需要 @机器人或使用特定指令前缀 |
| **权限要求** | 不同平台对机器人权限的要求不同，如消息读取权限、群管理权限等 |

具体差异请参考各频道的详细接入说明。

## 频道接入

JiuwenSwarm 支持多种聊天平台频道接入，根据地域分为以下两类：

### 国内频道

| 频道 | 说明 |
|------|------|
| [小艺](国内频道.md#小艺) | 华为鸿蒙生态智能助理，仅支持私聊 |
| [飞书](国内频道.md#飞书) | 企业协作平台，支持私聊和群聊，含数字分身功能 |
| [钉钉](国内频道.md#钉钉) | 企业协作平台，支持私聊和群聊 |
| [企业微信](国内频道.md#企业微信) | 企业即时通讯工具，支持私聊和群聊，含数字分身功能 |
| [个人微信](国内频道.md#个人微信) | 个人即时通讯工具，支持私聊 |

详细配置说明请参考：[国内频道](国内频道.md)

### 国外频道

| 频道 | 说明 |
|------|------|
| [Telegram](国外频道.md#telegram) | 国外即时通讯工具，支持私聊和群聊 |
| [Discord](国外频道.md#discord) | 游戏社区平台，支持私聊和群聊 |

详细配置说明请参考：[国外频道](国外频道.md)
